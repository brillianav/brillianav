## Working Principles

```txt
Make it clear.
Make it responsive.
Make it editable.
Make it maintainable.
```

- I prefer practical solutions over unnecessary complexity.
- I pay attention to the small details: spacing, mobile behavior, loading speed, and editor experience.
- I care about handoff, because a website should still make sense after the first launch.
